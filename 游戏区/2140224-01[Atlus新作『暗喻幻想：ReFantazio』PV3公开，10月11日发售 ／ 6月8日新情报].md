####  八佾舞于潭  
##### 746#       发表于 2024-9-21 01:47

我琢磨了一个宣发遇冷的理论：

P5 经过二次元提炼的日式城市生活呈现是极其稀缺的体验（没有调研，胡说的），在此基础上时髦 UI 强化了这个概念，主角团的战斗身份是怪盗，宫殿也多富有现代要素，因此时髦 UI 对传统 JRPG 部分也是增幅效应。

但暗喻幻想表面上是个大家都看腻的异世界奇幻，在此基础上引入奇幻版时髦 UI 并不会获得什么收益，经过了初期的名牌效应和 UI 新鲜感后，大众会暗喻幻想失去了兴趣，因为感觉不到它能提供什么稀缺的体验。

UI 设计上，虽然暗喻幻想跟 P5 一样大胆，但在具体构件上，则呈现奇幻与现代的区别，比如暗喻幻想多用彩雾和星图式的线条，而 P5 则是嘻哈涂鸦等常见的夸张几何要素，显然 P5 更容易让人迁移兴趣，而暗喻幻想的 UI 在震撼之于，只会让人不解。

人设上亦存在问题，JRPG 的人设，总是在诞生时代的基础上提供往前一步的幻想和尝试，但暗喻幻想不仅出现了大量 Persona 旧演员，在原生奇幻形象上，也仿佛是在追寻过去的梦。实际上真正具有精神的 JRPG 在暗喻幻想的舞台上，基本是要掏出加特林、暴走族卡车和北海道辣妹的。

但这只是引进门层面的考量，只要做的足够认真，收尾妥善，那么在现今的 JRPG 供应水准下，暗喻幻想终会是消费者最好的选择。
